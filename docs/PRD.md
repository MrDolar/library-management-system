# 图书管理系统 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
智能图书管理系统 (AI-Powered Library Management System)

### 1.2 产品定位
面向中小型图书馆、学校图书室、企业图书角的智能化图书管理平台，集成 AI 能力提供智能推荐、语义搜索、自动分类等功能。

### 1.3 目标用户
- 图书管理员
- 读者/借阅者
- 系统管理员

## 2. 核心功能

### 2.1 图书管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图书录入 | 支持手动录入、ISBN 扫描录入、批量导入 | P0 |
| 图书编辑 | 修改图书信息、封面、分类 | P0 |
| 图书删除 | 软删除，保留历史记录 | P0 |
| 图书检索 | 按书名、作者、ISBN、分类检索 | P0 |
| 库存管理 | 实时库存数量、库存预警 | P1 |

### 2.2 借阅管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 借书 | 记录借阅人、借阅时间、应还时间 | P0 |
| 还书 | 记录归还时间、自动计算逾期 | P0 |
| 续借 | 支持在线续借，限制续借次数 | P1 |
| 预约 | 图书在借时可预约，到书自动通知 | P2 |
| 逾期提醒 | 自动发送逾期通知 | P1 |

### 2.3 用户管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户注册 | 支持用户名/邮箱注册 | P0 |
| 用户登录 | JWT 认证 | P0 |
| 角色权限 | 管理员/普通用户分级权限 | P0 |
| 个人中心 | 借阅历史、个人信息管理 | P1 |

### 2.4 AI 智能功能 ⭐
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 智能推荐 | 基于借阅历史和偏好推荐图书 | P0 |
| 语义搜索 | 自然语言搜索，如"我想看一本关于太空的科幻小说" | P0 |
| 智能分类 | AI 自动为新书推荐分类标签 | P1 |
| 摘要生成 | 自动生成图书摘要/简介 | P1 |
| 对话助手 | AI 助手回答图书馆相关问题 | P2 |

### 2.5 数据统计
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 借阅统计 | 热门图书、借阅趋势 | P1 |
| 用户分析 | 活跃用户、阅读偏好 | P2 |
| 库存报表 | 库存状态、分类分布 | P1 |

## 3. AI 功能详细设计

### 3.1 智能推荐
- **输入**: 用户借阅历史、浏览记录、评分
- **输出**: 推荐图书列表（含推荐理由）
- **实现**: 基于内容的协同过滤 + LLM 增强
- **API**: OpenAI Compatible（支持多种 LLM 后端）

### 3.2 语义搜索
- **输入**: 用户自然语言查询
- **输出**: 匹配的图书列表
- **实现**: 文本 Embedding 向量相似度搜索
- **要求**: 图书元数据预计算 Embedding 并存储

### 3.3 智能分类
- **输入**: 图书标题、简介、目录
- **输出**: 推荐分类标签（置信度）
- **实现**: LLM Few-shot 分类

### 3.4 安全设计
- API Key 仅存储在服务端环境变量
- 前端不接触任何密钥
- 所有 AI 请求通过后端代理转发
- 支持 API Key 轮换

## 4. 非功能需求

### 4.1 性能
- 页面加载 < 2s
- 搜索响应 < 500ms
- AI 推荐响应 < 3s
- 支持 100 并发用户

### 4.2 安全
- HTTPS 强制
- JWT Token 认证
- SQL 注入防护
- XSS/CSRF 防护
- 敏感数据加密存储

### 4.3 可用性
- 响应式设计，支持移动端
- 无障碍访问支持
- 多语言支持（中/英）

## 5. 技术架构

### 5.1 前端
- React 18 + TypeScript
- Ant Design 组件库
- React Router 路由管理
- Axios HTTP 客户端

### 5.2 后端
- Python 3.11 + FastAPI
- SQLAlchemy ORM
- JWT 认证
- Pydantic 数据校验

### 5.3 数据库
- SQLite（开发/小规模）
- PostgreSQL（生产环境）

### 5.4 AI 集成
- OpenAI Compatible API
- 支持：OpenAI / DeepSeek / 本地模型
- 向量存储：FAISS / ChromaDB
